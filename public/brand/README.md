# Brand Asset Placeholders

Upload the finalized brand files here when they are approved:

- `brand-logo-primary.svg` (full lockup for headers)
- `brand-logo-symbol.svg` / `logo-mark.svg` (symbol-only mark for compact use)
- `brand-logo-monochrome.svg` (optional white or dark single-color variant)

Until the official assets are delivered, the app falls back to the mock logo files inside `src/assets/brand/`. Keeping these placeholder notes in `/public/brand` ensures static references resolve correctly on GitHub Pages without bundling temporary binaries.
