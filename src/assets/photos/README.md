# Photos & Team Moments

为 About / Culture 模块准备的建议文件：

- **photo-office-01.jpg**：团队在开放式办公区协作的照片，暖色调自然光。
- **photo-office-02.jpg**：会议或工作坊瞬间，展示产品讨论或用户访谈。
- **photo-event-01.jpg**：社区活动或外国人支援现场照，包含参与者互动。
- **photo-event-02.jpg**：企业培训或 AI スキルアップ研修的现场抓拍。
- **photo-team-portrait.jpg**：核心成员合影，浅色背景，利于在 about 页面展示。

推荐尺寸 1600px 以上，JPG 80% 品质，注意人物肖像授权。
