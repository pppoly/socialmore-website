# Brand Assets

以下为品牌识别所需的文件命名和建议视觉内容：

- **brand-logo-primary.svg**：主标志（英文 SOCIALMORE + 日文ロゴ）横版，用品牌渐变（#25B7B0 → #0F8AD7），透明背景。
- **brand-logo-horizontal-ja.svg**：日英双行排版的横向 Logo，左侧为三条曲线符号，右侧为日文社名，适合页眉。
- **brand-logo-symbol.svg**：仅包含三条渐变曲线的符号版本，可在 App Header 或 favicon 中使用。
- **brand-logo-monochrome.svg**：单色版本（白或深蓝），用于深色背景和印刷物。
- **brand-gradient-wave.svg**：以 Logo 曲线为元素的抽象波浪，可在 hero overlay 或背景中重复铺满。

> 以上文件将由设计/运营在交付最终视觉素材后上传到本目录。当前仓库附带了一个 `brand-logo-symbol.svg` 的渐变占位版本，方便开发和预览。
