# Icon Set

以下图标以 SVG 形式交付，描边 1.5px，方便在浅色/深色背景上使用：

- **icon-membership.svg**：会員管理 SaaS 图标，建议绘制为带有卡片/人像的抽象界面。
- **icon-community.svg**：コミュニティ／外国人支援图标，体现多语言对话气泡或握手元素。
- **icon-ai.svg**：企業向けAIスキルアップ图标，结合 AI 芯片 + 上升箭头。
- **icon-news.svg**：新闻/更新图标，可用喇叭或通知面板图形。
- **icon-contact.svg**：联系方式/联系表单图标，可用信封 + 点状连线。

所有 SVG 请保持可编辑路径，不嵌入位图。目前目录内的 `icon-*.svg` 为轻量示例，可直接替换成正式图标。
