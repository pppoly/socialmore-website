# Hero Visual Assets

- **hero-main.jpg**：ホーム／Landing Hero のメイン背景。1920x1080 前後、軽い被写界深度で東京の都市・コミュニティ風景をぼかし、ブランドのブルー〜オレンジのグラデーションライトを加える。
- **hero-overlay-shapes.svg**：前景に重ねる抽象線画。ブランドロゴの三本ラインを伸ばしたグリッド状のパスで、透明背景。
- **hero-cta-arrow.svg**（必要なら）：CTA ボタン付近に置く矢印またはモーションライン。極細ライン＋グラデーション。

> これらのファイルは hero セクション専用で、運営が最終画像をアップロードします。現状は `hero-main.svg` / `hero-overlay-shapes.svg` のシンプルなモックを入れてあります。
